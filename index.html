Phishing

El phishing es una técnica utilizada para engañar a las personas y robar información personal mediante correos electrónicos o sitios web falsos que imitan entidades legítimas.

¿Cómo identificarlo?

  • Correos con urgencia o amenazas
  • Enlaces sospechosos
  • Errores de ortografía

Recomendaciones

  • No ingresar datos en enlaces desconocidos
  • Verificar siempre la URL
  • Usar autenticación en dos factores

Estafas en WhatsApp

Los ciberdelincuentes utilizan WhatsApp para engañar a las personas mediante mensajes falsos.

Tipos comunes

  • Suplantación de identidad
  • Enlaces maliciosos
  • Solicitud de códigos de verificación

Cómo prevenir

  • No compartir códigos
  • No abrir enlaces sospechosos
  • Activar verificación en dos pasos

Contraseñas Seguras

Una contraseña segura protege tu información personal frente a accesos no autorizados.

Características de una buena contraseña

  • Mínimo 8 caracteres
  • Uso de símbolos y números
  • No usar datos personales

Buenas prácticas

  • No reutilizar contraseñas
  • Cambiar periódicamente
  • Usar gestores de contraseñas
